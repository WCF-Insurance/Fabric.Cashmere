The footer is set up to sit relative to other content on the page. In order to ensure that the footer sits at the bottom of the page, the div that holds the page content should be styled with a min-height attribute that calculates the height of the viewport (or the parent container) minus the combined height of the header and footer.

For example, with a header height of 72px and a footer height of 122px, the content container should be set to:

min-height: calc(100% - 194px);

Media queries may be used to keep footer content from running off the screen when the height of the footer changes. 

        <hc-header ... ></hc-header>

        <div class="page-content-footer">
        ...
        </div>

        <hc-footer ... ></hc-footer>
